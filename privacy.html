Политика конфиденциальности KoftaGard

Мы уважаем вашу приватность. KoftaGard не собирает и не хранит ваши личные данные без вашего согласия. Все действия с данными выполняются только по инициативе пользователя.

1. Обработка изображений

Пользователь может загружать изображения для распознавания объектов в них. Загруженные изображения обрабатываются ИИ, но не сохраняются на сервере без вашего согласия.

2. Использование данных

Загруженные изображения используются исключительно для предоставления результатов распознавания. Мы не передаём их третьим лицам.

3. Защита данных

Мы применяем меры безопасности для предотвращения несанкционированного доступа к загруженным изображениям во время обработки.

4. Изменения политики

Мы можем обновлять эту политику по мере необходимости. Рекомендуем периодически проверять этот раздел.
